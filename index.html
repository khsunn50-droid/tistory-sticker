티스토리 이미지 스티커

오른쪽 아래 패널에서 이미지를 업로드하고 스티커를 배치해보세요.
코드 복사 버튼을 누르면 티스토리에 바로 붙여넣을 수 있는 코드와 적용 방법을 볼 수 있어요.

사용법

조작법

이미지 업로드 → 썸네일 클릭으로 추가 (레이아웃 위 등장)
드래그로 이동 · + / − 로 크기 조절
뒤로 보내기 → 배경 / 앞으로 → 전경 복귀
배경 스티커 클릭 → 편집 툴바 팝업
고정 → 위치 잠금

스티커 관리
클릭하거나 이미지를 끌어다 놓으세요
PNG · JPG · GIF · WebP
업로드한 이미지가 여기에 표시됩니다

썸네일 클릭 → 레이아웃 위에 추가
뒤로 → 배경 이동 | 앞으로 → 전경 복귀
배경 스티커 클릭 → 편집 툴바 팝업
