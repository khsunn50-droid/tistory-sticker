티스토리 이미지 스티커
무료 · 관리자 전용
통합 코드 1개만 붙여넣으면 끝

블로그 배경에 이미지 스티커를
자유롭게 붙이세요

내 사진, GIF, PNG 투명 이미지를 배경에 배치하고
드래그로 이동, 크기 조절, 위치 고정까지 — 관리자에게만 보여요

아래 가이드를 따라 티스토리에 붙여넣으면 바로 사용할 수 있어요

적용 방법 총 3단계 · 약 3분

1

스킨 편집 화면 열기

티스토리 관리자 → 스킨 편집으로 이동해요

티스토리 관리자▶ 꾸미기▶ 스킨 편집▶ 우측 상단 "HTML 편집" 클릭
HTML 편집을 누르면 코드를 직접 수정할 수 있는 창이 열려요. 겁먹지 않아도 괜찮아요 — 정해진 곳에 붙여넣기만 하면 돼요!
2

통합 코드 붙여넣기

코드 한 개만 복사해서 정해진 위치에 붙여넣으면 돼요

1
위의 통합 코드 복사하기 버튼을 눌러 코드를 복사해요
2
HTML 편집 창에서 Ctrl + F (맥은 Cmd + F)를 눌러 검색창을 열고 </body> 를 검색해요
3
</body> 바로 앞에 커서를 놓고 복사한 코드를 Ctrl + V로 붙여넣어요
4
오른쪽 위 적용 버튼을 눌러 저장해요
꼭 </body> 바로 앞에 붙여야 해요. 다른 위치에 넣으면 작동하지 않을 수 있어요.
3

#wrap 에 CSS 한 줄 추가하기

스티커가 레이아웃 뒤로 제대로 들어가려면 이 작업이 꼭 필요해요

1
같은 HTML 편집 창에서 #wrap 또는 .wrap 이라는 CSS를 찾아요
스킨마다 이름이 달라요. 없으면 #container 나 #content 를 찾아보세요
2
그 안에 아래 두 줄을 추가해요
CSS — #wrap 안에 추가 
position: relative;
z-index: 10;
이 설정이 없으면 스티커를 배경으로 보냈을 때 레이아웃 뒤로 안 들어가고 겹쳐 보여요. 꼭 추가해 주세요!

이제 끝이에요!

블로그에 접속하면 오른쪽 아래에 스티커 패널이 보여요.
관리자 계정으로 로그인했을 때만 패널이 표시되고,
방문자에게는 배치된 스티커만 보여요.

기능 안내

+
이미지 추가 — 패널에서 이미지 업로드 후 썸네일 클릭 → 레이아웃 위에 등장
↔
위치 이동 — 스티커를 드래그해서 원하는 곳으로
±
크기 조절 — 툴바의 + / − 버튼으로 크기 조절
↓
뒤로 보내기 — "뒤로" 버튼 → 배경 레이어로 이동
↑
앞으로 꺼내기 — 배경 스티커 클릭 → 툴바에서 "앞으로" 버튼
📌
고정 — 실수로 움직이지 않도록 위치를 잠글 수 있어요
✕
삭제 — 개별 삭제 또는 전체 삭제
PNG 투명 배경 이미지를 쓰면 훨씬 자연스럽게 배치돼요. 스티커 위치는 새로고침하면 초기화되니 마음에 드는 배치가 완성되면 페이지를 닫지 마세요!

파이어베이스 연동 — 새로고침해도 스티커 위치 유지하기 선택 사항 · 약 10분

기본 코드만 쓰면 새로고침할 때마다 스티커 위치가 초기화돼요. 파이어베이스를 연동하면 스티커 위치·크기 정보가 구글 서버에 저장되어 새로고침해도 그대로 유지됩니다. 완전 무료예요!
1

파이어베이스 프로젝트 만들기

구글 계정으로 파이어베이스에 로그인하고 새 프로젝트를 생성해요

1
console.firebase.google.com 에 접속해서 구글 계정으로 로그인해요
2
프로젝트 추가 또는 + 새 프로젝트 버튼을 클릭해요
3
프로젝트 이름을 입력하고 (예: my-blog-sticker) 계속을 눌러요. Google 애널리틱스는 꺼도 무관해요
4
프로젝트 만들기를 눌러 완료해요. 잠깐 기다리면 프로젝트 대시보드로 이동해요
2

Realtime Database(실시간 데이터베이스) 만들기

스티커 위치 데이터를 저장할 데이터베이스를 생성해요

1
왼쪽 메뉴 상단 검색창에 Realtime Database를 검색하고 클릭해요
2
데이터베이스 만들기 버튼을 클릭해요
3
위치는 기본값 그대로 두고 다음을 눌러요
4
보안 규칙에서 테스트 모드에서 시작을 선택하고 실행을 눌러요
5
데이터베이스가 생성되면 상단 규칙 탭을 클릭하고, 기존 내용을 지운 뒤 아래 코드를 붙여넣은 후 게시를 눌러요
Firebase Rules — 규칙 탭에 붙여넣기 
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
게시 버튼을 꼭 눌러야 해요! 누르지 않으면 규칙이 적용되지 않아요.
3

내 앱 등록하고 연결 정보 확인하기

코드에 넣을 나만의 연결 정보(Key)를 발급받아요

1
왼쪽 상단 프로젝트 개요로 이동해서, 화면 중앙의 </> (웹) 아이콘을 클릭해요
2
앱 닉네임을 입력하고 (예: blog) 앱 등록을 눌러요
3
아래처럼 firebaseConfig 코드가 나타나요. 여기서 이 4가지 값을 복사해 메모장에 저장해두세요
apiKey authDomain databaseURL projectId
4
databaseURL이 안 보이면 왼쪽 메뉴 Realtime Database 상단에 있는 https://프로젝트명.firebaseio.com 주소를 직접 복사해서 쓰면 돼요
4

파이어베이스 저장 코드 붙여넣기 — 가장 중요!

기존 통합 코드 대신 아래 코드를 사용하고, 내 정보로 수정해요

1
아래 파이어베이스 연동 코드 복사하기 버튼을 눌러 코드를 복사해요
2
코드 맨 위쪽에 본인의_API_KEY 등 4곳을 3단계에서 확인한 값으로 교체해요
3
티스토리 HTML 편집에서 기존에 붙여넣은 통합 코드를 삭제하고, 이 코드로 교체해서 붙여넣어요
4
적용을 눌러 저장해요. 이제 스티커를 배치하면 자동으로 서버에 저장돼요!
스티커 위치(x, y)와 크기만 저장해요. 이미지 파일 자체는 저장되지 않으므로 매번 업로드가 필요해요.
