티스토리 이미지 스티커
비밀번호 보호

블로그 배경에 이미지 스티커를
자유롭게 붙이세요

내 사진, GIF, PNG 이미지를 배경에 배치하고
드래그 이동, 크기 조절까지 — 비밀번호로 보호돼요

아래 가이드를 따라 티스토리에 붙여넣으면 바로 사용할 수 있어요

적용 방법

1

스킨 편집 화면 열기

티스토리 관리자 → 꾸미기 → 스킨 편집 → HTML 편집

2

통합 코드 붙여넣기

</s_t3> 바로 앞에 붙여넣으세요

1
위의 통합 코드 복사하기 버튼을 눌러요
2
HTML 편집 창에서 Ctrl+F로 </s_t3> 를 검색해요
3
</s_t3> 바로 앞에 커서를 놓고 Ctrl+V로 붙여넣어요
4
적용 버튼을 눌러 저장해요
3

비밀번호 설정하기

코드 안의 비밀번호를 본인만 아는 것으로 바꾸세요

코드 안에서 이 부분을 찾아서 원하는 비밀번호로 바꿔주세요

var TIS_PASSWORD = '1234';
비밀번호는 같은 탭에서는 한 번만 입력하면 돼요. 탭을 닫으면 다시 물어봐요.

이제 끝이에요!

오른쪽 아래 파란 자물쇠 버튼을 클릭하고
비밀번호를 입력하면 스티커 패널이 열려요.
방문자에게는 자물쇠 버튼만 보이고 패널은 열리지 않아요.

파이어베이스 연동 — 새로고침해도 스티커 위치 유지하기 선택 · 약 10분

기본 코드만 쓰면 새로고침 시 스티커 위치가 초기화돼요. 파이어베이스를 연동하면 서버에 저장되어 유지됩니다. 완전 무료예요!
1

파이어베이스 프로젝트 만들기

1
console.firebase.google.com 에서 구글 로그인
2
프로젝트 추가 → 이름 입력 (예: my-blog-sticker) → 생성
2

Realtime Database 만들기

1
왼쪽 메뉴 → Realtime Database → 데이터베이스 만들기
2
위치 기본값, 테스트 모드에서 시작 → 실행
3
규칙 탭 → 아래 내용으로 교체 → 게시

Firebase Rules

{
  "rules": {
    ".read": true,
    ".write": true
  }
}
3

앱 등록하고 연결 정보 확인하기

1
프로젝트 개요 → </> (웹) 아이콘 → 앱 닉네임 입력 → 등록
2
아래 4가지 값을 메모장에 복사해두세요
apiKeyauthDomaindatabaseURLprojectId
4

코드에 Firebase 정보 입력하기

1
스킨 HTML에 붙여넣은 코드에서 아래 부분을 찾아요

이 부분을 찾아서 본인 정보로 교체하세요

var TIS_FB_CONFIG = {
  apiKey: "본인의_API_KEY",
  authDomain: "본인의_PROJECT_ID.firebaseapp.com",
  databaseURL: "https://본인의_PROJECT_ID-default-rtdb.firebaseio.com",
  projectId: "본인의_PROJECT_ID"
};

기능 안내

+
이미지 추가 — 패널에서 이미지 업로드 후 썸네일 클릭
↔
위치 이동 — 스티커를 드래그해서 원하는 곳으로
±
크기 조절 — 툴바의 + / − 버튼
↓
뒤로 보내기 — 배경 레이어로 이동, 방문자에게도 보여요
↑
앞으로 꺼내기 — 배경 스티커 클릭 → 툴바에서 “앞으로”
×
삭제 — 개별 삭제 또는 전체 삭제
